# 🧩 Sudoku em Java

Projeto desenvolvido como parte do desafio Bradesco Java Cloud Native, com o objetivo de aplicar conceitos de lógica de programação e Programação Orientada a Objetos em Java.

## 🚀 Funcionalidades

- Geração e execução de jogo Sudoku
- Sistema de verificação de jogadas
- Timer de tempo de jogo
- Sistema de dicas para auxiliar o jogador
- Interface em console

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Estruturas de dados
- Lógica de programação

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram aplicados conceitos como:

- Estruturação de código em Java
- Manipulação de matrizes
- Validação de regras do Sudoku
- Organização e modularização de código
- Boas práticas de programação

## 💡 Melhorias futuras

- Interface gráfica (GUI)
- Níveis de dificuldade
- Sistema de ranking de jogadores
- Salvamento de partidas
